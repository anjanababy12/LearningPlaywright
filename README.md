# Learning Playwright

This repository contains code and examples for learning Playwright for test automation.

## Project Structure

- **Chapter1_basics**: Contains basic examples and setup verification scripts.
  - `1_Basics.js`: Introduction to basic Playwright concepts.
  - `2_Stepbystep.js`: Step-by-step guide code.
  - `3_verifysetup.js`: Script to verify the Playwright setup.
  - `4_hotcode.js`: Additional examples.

## Getting Started

1.  Clone the repository:
    ```bash
    git clone https://github.com/anjanababy12/LearningPlaywright.git
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Run the tests (example):
    ```bash
    npx playwright test
    ```
